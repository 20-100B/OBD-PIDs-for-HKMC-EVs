How to import the Dashboards manually.



All the .csv files in your car's extended pids folder need to be copied onto the same folder in the phone. These files need to be added into Torque. - Setting up Torque Pro to show OBD data on Hyundai / Kia Electric Vehicles Then the dashboard you want needs to be copied onto the same folder in the phone. And imported into Torque.
